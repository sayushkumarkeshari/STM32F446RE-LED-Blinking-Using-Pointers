# STM32F446RE LED Blinking Using Pointers

This project demonstrates LED blinking on STM32F446RE using direct register-level programming and memory-mapped pointers in Embedded C.

## Features
- GPIOA clock enable
- PA5 output configuration
- Direct register access using pointers
- Software delay implementation
- Bare-metal programming

## Microcontroller
STM32F446RE

## Registers Used
- RCC_CR
- RCC_AHB1ENR
- GPIOA_MODER
- GPIOA_OTYPER
- GPIOA_SPEEDR
- GPIOA_ODR

## Development Environment
- Keil uVision
- Embedded C
